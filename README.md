# LED-Pattern-Generator-using-Speed-Controller
A Verilog-based digital system designed to generate dynamic LED light patterns with real-time speed control. The project implements customizable sequential lighting effects (such as chasing, blinking, or shifting patterns) whose execution rate can be adjusted dynamically using manual speed controls
